**Question #2:**
How can you remove duplicates from this list:

[1,2,2,2,3,3,4,4,5]

**Answer #2:**
By making it a set and then a list again: e.g.

list1 = [1,2,2,2,3,3,4,4,5]
list1 = list(set(list1))
